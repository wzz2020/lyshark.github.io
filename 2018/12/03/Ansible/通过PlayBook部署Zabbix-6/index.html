通过PlayBook部署Zabbix(6)

发表于 2018-12-03 | 更新于 2018-12-09 | 分类于 Ansible

Ansible是新出现的自动化运维工具,基于Python开发,集合了众多运维工具（puppet、cfengine、chef、func、fabric）的优点,实现了批量系统配置、批量程序部署、批量运行命令等功能,ansible是基于模块工作的,本身没有批量部署的能力,真正具有批量部署的是ansible所运行的模块,ansible只是提供一种框架.

Zabbix 是目前最流行的一款企业级别的监控软件,它的易用性以及扩展性非常强大,加上自带丰富的 API 功能,使得它儿乎成为每个互联网公司的标配运维组件. Zabbix 的默认架构是 C/S 架构,当然 zabbix 也支持 Proxy 代理,本篇文章将介绍如何使用PlayBook剧本的方式部署Zabbix组件.
