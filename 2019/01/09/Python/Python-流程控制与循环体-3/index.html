双分支结构: 双分支用来判断两种不同的情况,比如要么为真要么为假.

多分支结构: 多分枝结构用来判断大多数情况,其给予每个结果一个返回值.

跳过执行与跳出语句

pass语句: pass是为了保持程序结构的完整性,该语句不做任何事情,一般用做占位语句.

break语句: break语句用来终止循环语句,即使循环没有结束任然要执行.

continue语句: 该语句语句用来告诉Python跳过当前循环的剩余语句,然后继续进行下一轮循环.
